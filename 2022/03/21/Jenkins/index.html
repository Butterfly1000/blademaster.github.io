运行registry

  • -i: 交互式操作。
  • -t: 终端。
  • -v: 将宿主机目录挂载到容器里, 或者说把镜像路径映射到本机。-v /宿主机目录:/容器目录
  • -d 指定容器的运行模式，后台运行，默认不会进入容器
  • -P (大写):是容器内部端口随机映射到主机的端口。
  • -p(小写) : 是容器内部端口绑定到指定的主机端口。-p 指定的主机端口 容器内部端口
  • docker attach
  • docker exec：推荐大家使用 docker exec 命令，因为此退出容器终端，不会导致容器的停止。
