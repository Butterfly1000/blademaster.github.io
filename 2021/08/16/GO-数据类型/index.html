变量声明语法

go可使用var关键字声明全局变量,但是:=这种方式是不能用在全局变量中的.:=只能用在函数体内部.

go函数体内的变量是如果被声明了,那么必须使用,不然会报错。注:全局变量声明未使用不会报错

变量初始化

Go 语言在声明变量时会默认给变量赋个当前类型的空值/零值

在方法中声明一个临时变量并赋初值

当一个变量或者新值被创建时, 如果没有为其明确指定初始值,go语言会自动初始化其值为此类型对应的零值。

类型 零值
数值类型 0
布尔类型 false
字符串 “”（空字符串）
slice nil
map nil
指针 nil
函数 nil
接口 nil
信道 nil
